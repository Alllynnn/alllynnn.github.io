## 📝 友链申请

如果要申请友链，请按照以下格式在评论区留言：

```yaml
- name: Alyn's blog    # 网站名称
  link: https://nikoioi.github.io/    # 网站链接
  avatar: your-avatar-url    # 头像链接
  descr: 生活与技术的分享    # 网站描述
```

### 申请要求

1. 网站要有实质性内容，建议已经发布 10 篇以上文章
2. 网站要有独立域名
3. 网站要有 HTTPS
4. 网站要有头像和网站图标
5. 网站要有评论系统
